This is the public portion of the ne7 package.

NE7 is a collection of computational utilities for neural data processing. 
The library is designated for agile development and undergoes frequent changes.
If a particular tool becomes sufficiently stable, it is typical moved into a more 
permanent. 

+dsp         -- general digital signal processing
+ip          -- general image processing
+info        -- general information theory
+stats       -- general statistics
+bayes       -- bayesian models 
+micro       -- analysis of microscopy data
+scanimage   -- utilities to process scanimage files (see scanimage.org)
+ephys       -- electrophysiology 
+rf          -- receptive field computations
+mat         -- general matlab helper functions
+population  -- neural population analysis
